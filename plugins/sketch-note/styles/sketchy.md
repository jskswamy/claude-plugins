# Sketchy Style

Medium roughness for a balanced appearance that's approachable yet readable.

## Excalidraw Properties

```json
{
  "roughness": 1,
  "fillStyle": "hachure",
  "strokeStyle": "solid"
}
```

## Characteristics

- **Roughness:** 1 (medium)
- **Lines:** Slightly imperfect, controlled wobble
- **Fill:** Hachure with tighter lines
- **Feel:** Professional yet approachable

## Best For

- Technical documentation
- Architecture overviews
- Team presentations
- Tutorial diagrams
- Readable sketches

## Color Recommendations

Use balanced, professional colors:
- Primary: `#a5d8ff` (light blue)
- Secondary: `#e9ecef` (light gray)
- Accent: `#d0bfff` (light purple)
- Background: `#ffffff` (white)

## Font Pairing

- **Labels:** fontFamily 1 (normal sans-serif)
- **Code/Data:** fontFamily 2 (monospace)
- **Headers:** fontFamily 3 (handwriting) for titles only

## Example Element

```json
{
  "type": "rectangle",
  "roughness": 1,
  "fillStyle": "hachure",
  "strokeWidth": 2,
  "strokeColor": "#1e1e1e",
  "backgroundColor": "#a5d8ff"
}
```
