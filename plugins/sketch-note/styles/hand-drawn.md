# Hand-Drawn Style

Maximum roughness for an authentic, hand-sketched appearance.

## Excalidraw Properties

```json
{
  "roughness": 2,
  "fillStyle": "hachure",
  "strokeStyle": "solid"
}
```

## Characteristics

- **Roughness:** 2 (maximum)
- **Lines:** Wobbly, imperfect
- **Fill:** Hachure (diagonal hand-drawn lines)
- **Feel:** Casual, brainstorming, informal

## Best For

- Brainstorming sessions
- Early-stage ideation
- Casual meeting notes
- Personal sketches
- Informal documentation

## Color Recommendations

Use warm, organic colors:
- Primary: `#a5d8ff` (soft blue)
- Secondary: `#b2f2bb` (soft green)
- Accent: `#ffec99` (warm yellow)
- Background: `#faf8f5` (cream) or `#ffffff` (white)

## Font Pairing

- **Labels:** fontFamily 3 (Virgil handwriting)
- **Code/Data:** fontFamily 2 (monospace)
- **Emphasis:** Bold stroke width

## Example Element

```json
{
  "type": "rectangle",
  "roughness": 2,
  "fillStyle": "hachure",
  "strokeWidth": 2,
  "strokeColor": "#1e1e1e",
  "backgroundColor": "#a5d8ff"
}
```
