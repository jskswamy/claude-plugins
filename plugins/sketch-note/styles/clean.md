# Clean Style

Zero roughness for crisp, professional diagrams.

## Excalidraw Properties

```json
{
  "roughness": 0,
  "fillStyle": "solid",
  "strokeStyle": "solid"
}
```

## Characteristics

- **Roughness:** 0 (none)
- **Lines:** Precise, straight
- **Fill:** Solid color (no hachure)
- **Feel:** Professional, polished, formal

## Best For

- Final documentation
- Client presentations
- Technical specifications
- Published diagrams
- Formal reports

## Color Recommendations

Use clean, corporate colors:
- Primary: `#a5d8ff` (light blue)
- Secondary: `#e9ecef` (light gray)
- Accent: `#1971c2` (darker blue for emphasis)
- Background: `#ffffff` (white)

## Font Pairing

- **Labels:** fontFamily 1 (normal sans-serif)
- **Code/Data:** fontFamily 2 (monospace)
- **Headers:** fontFamily 1 with larger fontSize

## Example Element

```json
{
  "type": "rectangle",
  "roughness": 0,
  "fillStyle": "solid",
  "strokeWidth": 2,
  "strokeColor": "#1e1e1e",
  "backgroundColor": "#a5d8ff"
}
```

## Stroke Width Recommendations

For clean style, consider thinner strokes:
- **Thin (1):** Detailed, delicate diagrams
- **Medium (2):** Standard clarity
- **Bold (4):** Emphasis, presentations
